# Notes

This directory is for scratch notes, experiments, brainstorming, and temporary working files.

## What Goes Here

- Brainstorming sessions
- Meeting notes
- Decision logs
- Experiments and explorations
- Quick references
- Temporary analysis

## Format

No strict format required - this is your working space. Use whatever structure helps you think.

## Examples

- `2026-02-12_tech-stack-comparison.md` - Comparing options
- `2026-02-13_meeting-notes.md` - Team discussion notes
- `experiment-babysitter-flow.md` - Testing out a feature idea
- `quick-refs.md` - Links and snippets you need frequently
